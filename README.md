強力手電筒第四代
===
### 本手電筒搭載50W LED燈板，亮度超過5000流明，透過4顆鋰電池供電，續航力長，提供長時間高亮度照明。因高功率LED發熱量大，本手電筒還配有散熱風扇，保證長時間使用不過熱，保證安全。外型設計為傳統手電筒的筒形機身，方便握持也方便收納。
